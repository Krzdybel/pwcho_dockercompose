## Uruchomienie

```
git clone https://github.com/Krzdybel/pwcho_dockercompose.git
cd pwcho_dockercompose
docker-compose up
```

Po wykonaniu komnedy docker-compose up i wystartowaniu kontenerow nalezy przejsc na adres ```localhost:6666```.
